# Академия Будущего - Презентация Маркетинговой Стратегии

## Описание

Это интерактивная презентация комплекса маркетинговых коммуникаций для вымышленной образовательной платформы "Академия Будущего". Презентация разработана в формате локального веб-сайта с современным дизайном и анимациями, предназначена для привлечения потенциальных студентов и демонстрации стратегии выхода на рынок.

## Особенности

- **Интерактивная навигация**: Переход между слайдами с помощью кнопок, индикаторов или клавиатуры (стрелки влево/вправо).
- **Авто-переход**: Слайды меняются автоматически каждые 8 секунд (пауза по пробелу).
- **Современный дизайн**: Градиенты, анимации, адаптивность для мобильных устройств.
- **Завлекательный контент**: Testimonials, статистика, кликабельные элементы для вовлечения аудитории.

## Структура проекта

- `index.html` - Основной файл презентации с HTML, CSS и JavaScript.

## Как запустить

1. Убедитесь, что у вас установлен Python 3.
2. Клонируйте или скачайте репозиторий.
3. Перейдите в директорию проекта:
   ```
   cd /path/to/project
   ```
4. Запустите локальный HTTP-сервер:
   ```
   python3 -m http.server 8000
   ```
5. Откройте браузер и перейдите по адресу:
   ```
   http://127.0.0.1:8000
   ```
   или
   ```
   http://localhost:8000
   ```

## Стратегия маркетинговых коммуникаций

Презентация включает следующие разделы:

1. **Введение** - Приветствие и цель презентации.
2. **Кто мы?** - Характеристика "Академии Будущего".
3. **Наши цели** - Коммуникационные цели (кликабельные для деталей).
4. **Наши сообщения** - Ключевые слоганы и фразы.
5. **Наши инструменты** - Маркетинговые инструменты (реклама, контент, email и т.д.).
6. **Каналы коммуникации** - Онлайн и оффлайн каналы.
7. **Ожидаемые результаты** - Метрики успеха и testimonials.
